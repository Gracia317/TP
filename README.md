Mini projects (for getting into C langage)

fichier2.c: A C program writes the first 50 prime numbers to a file, then displays only those whose line number is divisible by 3. 
            Next, it changes all occurrences of 5 to V. 
            Finally, it displays the numbers in reverse order; for example: 5, 12, 57, ..., 79, 231 becomes 132, 97, ..., 7V, 21, V

gestioninscription.c: A C program that manages student registration, writing information to a file (last name, first name(s), date of birth, address, level). 
                      Sorting options include alphabetical order and ascending order by date of birth. 
                      Options also include pattern search, deletion, and modification.

polynome.c:A C program that calculates the image of any function entered as an argument (polynomial, logarithm, exponential, trigonometric, etc). 
            For example, `./polynome.exe y=2x²+5x³-1` for x=-3 or `./polynome.exe y=log(sinx)` for x=1
